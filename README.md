# Club Hive - Student Club Management System

Club Hive is a web based platform designed to manage student club operations, including membership management, event coordination, communication, and activity tracking.

The project was developed as part of a team effort, with primary responsibility for system design, backend development, dashboards, and deployment.

---

## Overview

Student clubs often rely on fragmented tools for managing members, events, and communication. Club Hive aims to centralize these workflows into a single platform with role-based access and structured data management.

The system focuses on simplifying administrative workflows while providing visibility into club activity through basic analytics.

---

## Core Features

- Role-based access for students, administrators, and faculty
- Club membership and event management
- Real-time notifications and updates
- Activity tracking and dashboard-level analytics
- Gamified elements to encourage participation and engagement within clubs
- Backend APIs with database-backed persistence

---

## Tech Stack

- **Backend:** Node.js (Express)
- **Frontend:** React
- **Database:** PostgreSQL
- **Caching:** Redis
- **Deployment:** Docker
